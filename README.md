# Active Directory Automation

Projeto com scripts PowerShell para automação de tarefas administrativas no Active Directory.

## Objetivo

Automatizar tarefas comuns de administração em ambientes corporativos utilizando PowerShell.

## Funcionalidades

- Criação automática de usuários
- Reset de senha
- Listagem de usuários
- Gerenciamento de grupos
- Auditoria básica de contas

## Tecnologias

- PowerShell
- Windows Server
- Active Directory

## Estrutura do projeto

scripts/
create-user.ps1
reset-password.ps1
list-users.ps1
